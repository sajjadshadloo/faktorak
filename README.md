# فاکتورک (Faktorak)

**افزونه فاکتور، برچسب پستی و مدیریت سفارشات برای ووکامرس**

## معرفی

فاکتورک یک افزونه حرفه‌ای برای ووکامرس است که امکانات چاپ فاکتور، برچسب پستی، فاکتور پیش‌فاکتور، خروجی سفارشات و مدیریت کامل اسناد را در پنل مدیریت وردپرس فراهم می‌کند.

---

## امکانات

### 🖨️ چاپ فاکتور و برچسب پستی
- چاپ فاکتور و برچسب پستی در فرمت A4
- دو قالب فاکتور: **کلاسیک** و **مدرن**
- تصویر امضای فروشنده روی فاکتور
- نمایش یادداشت مشتری در هر دو قالب
- QR Code محلی (بدون سرویس خارجی)

### 📄 پیش‌فاکتور
- وضعیت سفارش اختصاصی برای پیش‌فاکتور
- دکمه تبدیل پیش‌فاکتور به سفارش قابل پرداخت از پنل ادمین
- تبدیل خودکار نوع سند پس از پرداخت موفق

### 📝 فاکتور دستی
- ساخت فاکتور یا پیش‌فاکتور دستی از محصولات دلخواه در wp-admin
- جستجوی AJAX برای محصول و مشتری با تکمیل خودکار اطلاعات مشتری
- صفحه **لیست فاکتورها** با قابلیت جستجو، فیلتر و حذف
- کپی سریع لینک مشتری و لینک پرداخت
- دکمه پرداخت درون فاکتور برای پیش‌فاکتورهای پرداخت‌نشده
- محافظت Token-based از لینک‌های فاکتور

### 📦 خروجی سفارشات *(جدید در ۱.۵)*
- خروجی CSV و Excel از سفارش‌ها
- خروجی PDF دسته‌جمعی
- خروجی ZIP شامل فاکتور جداگانه برای هر سفارش
- انتخاب دستی سفارش‌ها برای خروجی
- فیلتر بر اساس وضعیت، تعداد و بازه تاریخ
- انتخاب اطلاعات دلخواه در خروجی

### 🗺️ نقشه در صفحه پرداخت
- امکان انتخاب موقعیت تحویل روی نقشه در چک‌اوت
- ذخیره موقعیت در متای سفارش
- نمایش لینک Google Maps در پنل ادمین

### ⚡ یکپارچه‌سازی
- ستون دسترسی سریع (فاکتور + برچسب پستی) در لیست سفارشات
- سازگاری کامل با **HPOS** (High-Performance Order Storage)
- شورت‌کدها: `[faktorak_invoice_button]` و `[faktorak_proforma_button]`
- تایپوگرافی یکپارچه IRANYekan در تمام صفحات پنل

---

## نصب

1. پوشه افزونه را در `wp-content/plugins/` آپلود کنید.
2. افزونه را از پنل مدیریت وردپرس فعال کنید.
3. به منوی **فاکتورک** در wp-admin بروید و تنظیمات را پیکربندی کنید.

---

## راهنمای استفاده

| کاربرد | روش |
|--------|-----|
| چاپ فاکتور/برچسب سفارش | باز کردن سفارش → متاباکس فاکتورک |
| ساخت فاکتور دستی | فاکتورک ← فاکتور دستی |
| مدیریت فاکتورها | فاکتورک ← لیست فاکتورها |
| خروجی سفارشات | فاکتورک ← خروجی سفارشات |
| دکمه فاکتور برای مشتری | شورت‌کد `[faktorak_invoice_button]` |

## تغییرات

### نسخه ۱.۵.۰
- اضافه شدن سیستم کامل خروجی سفارشات (CSV، Excel، PDF، ZIP)
- امکان انتخاب دستی سفارش‌ها و فیلتر پیشرفته
- سازگاری کامل با HPOS ووکامرس
- یکپارچه‌سازی ظاهر پیشخوان فاکتورک
- بهبود UI صفحات تنظیمات، فاکتور دستی، لیست فاکتورها
- بهینه‌سازی CSS و جلوگیری از کش شدن تغییرات

### نسخه ۱.۴.۰
- فاکتور دستی با جستجوی AJAX
- ستون دسترسی سریع در لیست سفارشات
- محافظت Token-based از لینک‌های فاکتور
- دکمه پرداخت درون فاکتور
- تبدیل خودکار پیش‌فاکتور به فاکتور رسمی

### نسخه ۱.۳.۱
- بهبودها و سخت‌سازی امنیتی
